# Investment Growth Calculator

A Python-based financial calculator designed to model long-term investment growth using compound interest and recurring contributions.

## Overview

This project allows users to input their own investment assumptions, including:
- Initial investment amount
- Monthly contributions
- Expected annual return
- Investment duration

The calculator then estimates portfolio growth over time and visualises the investment performance using Python graphs.

## Technologies Used

- Python
- Matplotlib

## Features

- User-defined investment inputs
- Compound interest calculations
- Monthly contribution modelling
- Portfolio growth visualisation
- Long-term investment forecasting
- Graphical display of portfolio value over time

## Financial Concepts Used

- Compound interest
- Investment growth
- Future value calculations
- Long-term investing
- Contribution modelling

## Example Inputs

- Initial Investment: £5,000
- Monthly Contribution: £200
- Annual Return: 7%
- Investment Period: 30 Years

## Example Outputs

The program calculates:
- Total contributions
- Investment growth
- Final portfolio value

and generates a graph showing portfolio growth over time.
